# Team Knowledge Base

Shared knowledge repository powered by [TeamDNA](https://github.com/xxx/teamdna).

## Structure

- `pitfalls/` — Lessons learned and debugging records
- `standards/` — Team conventions and coding standards
- `solutions/` — Proven solutions and architecture decisions

## Usage

```bash
/dna-search <keyword>   # Search knowledge base
/dna-push               # Share new experience
/dna-pull               # Sync latest knowledge
/dna-index              # Rebuild index
```
